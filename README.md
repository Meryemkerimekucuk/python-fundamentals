# Miuul AI Data Scientist Bootcamp - Python Basics

Bu depo, **Miuul AI Data Scientist Bootcamp** kapsamında yürüttüğüm veri bilimi yolculuğumun ilk adımı olan temel Python programlama notlarını ve uygulamalarını içermektedir.

Veri analizinin ve yapay zekanın temelini oluşturan algoritmik düşünce ve veri yönetimi mantığını pekiştirmek adına hazırladığım ilk hafta çalışmalarımı burada derledim.

---

## 📚 Hafta İçeriği

Bu hafta repoya eklenen kod notları, Python dilinin temel yapı taşlarını kapsamaktadır:

* **Temel Python Yapısı:** Değişkenler, temel fonksiyonlar ve kod yazım esasları.
* **Veri Tipleri (Data Types):** Sayısal (int, float), metinsel (string) ve mantıksal (boolean) yapılar.
* **Koşullu İfadeler (Conditions):** Program akışını kontrol eden `if - elif - else` blokları.
* **Döngüler (Loops):** Tekrarlı işlemleri yöneten `for` ve `while` yapıları.
* **Veri Yapıları (Data Structures):** Verileri organize etmek için kullanılan **List**, **Dictionary**, **Tuple** ve **Set** yapıları ile temel metotları.
